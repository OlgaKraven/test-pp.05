# Театр — MVP (Модули 1–3)

Готовый учебный проект по предметной области «театр».

## Быстрый старт

```bash
python -m venv .venv && . .venv/bin/activate
pip install -r requirements.txt
python app.py
```
Откройте http://127.0.0.1:5000
